Installing Anki to learn vocabulary

1. Install Anki from official website and Ankidroid from PlayStore

2. Download and import pre-made deck

Option:

  • Kaishi 1.5K [English] [Bahasa Indonesia]
  • Core 2.3k
  • Core 6k [Part 1] [Part 2]

3. Configure Anki settings

  • Turn on FSRS
  • Set "Daily Limits" to 9999
